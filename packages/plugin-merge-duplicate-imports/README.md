## Install

```
npm i @putout/plugin-merge-duplicate-imports
```

## Rule

```json
{
    "rules": {
        "merge-duplicate-imports": "on"
    }
}
```

## ‚ùå Example of incorrect code

```js

```

## ‚úÖ Example of correct code
