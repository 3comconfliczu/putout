üêä[`Putout`](https://github.com/coderaiser/putout) plugin adds ability to remove useless `conditions`.

## Install

```
npm i @putout/plugin-remove-useless-conditions
```

## Rule

```json
{
    "rules": {
        "remove-useless-conditions": "on"
    }
}
```
